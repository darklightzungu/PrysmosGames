# HuntrixRainbowObby_v1

Kid-safe rainbow carpet obby inspired by the Huntrix inbox concept.

- **Audience:** ~5 years old — no combat, soft respawns, cheerful copy
- **Mechanics:** checkpoint platforms, star collectibles, Space/A flying carpet boost
- **Monetisation:** rainbow trail GamePass (`GAMEPASS_ID_PLACEHOLDER`)
